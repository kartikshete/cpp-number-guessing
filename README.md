# 🎮 CPP Number Guessing Game

 ![React](https://img.shields.io/badge/-React-blue?style=flat-square) ![Node.js](https://img.shields.io/badge/-Node.js-blue?style=flat-square) ![Productivity](https://img.shields.io/badge/-Productivity-blue?style=flat-square) ![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)

An interactive, browser-based Number Guessing logic game wrapped in a highly polished React execution utilizing modern effects.

## 📝 Overview
Who said classic logic games have to look boring? This web adaptation of the classic number-guessing algorithm uses React and Vite to deliver a blazing-fast experience. Enhanced with `canvas-confetti` and `framer-motion`, this game turns simple mathematical deducing into a visually rewarding puzzle challenge.

## ✨ Key Features
- **High Performance React Engine**: Minimal component re-renders ensures logic checks are instant.
- **Rewarding UX**: Integrated confetti bursting physics via `canvas-confetti` when the correct number is hit.
- **Fluid UI**: `framer-motion` adds sleek, bounce-driven animations and layout transitions.
- **Lucide Iconography**: Clean UI indicators generated through scalable `lucide-react` SVGs.

## 🛠 Tech Stack
- Framework: `React 19`
- Bundler/Compiler: `Vite` (with SWC plugin for lightning-fast compilation)
- Animations: `Framer Motion` & `Canvas Confetti`

## 🚀 Getting Started

```bash
# Enter project directory
cd cpp-number-guessing

# Install dependencies rapidly 
npm install

# Instantly fire up the Vite dev server
npm run dev
```


## 🌐 Deployment

### Vercel / Netlify (Recommended)
1. Push this repository to GitHub.
2. Connect your GitHub account to [Vercel](https://vercel.com) or [Netlify](https://www.netlify.com).
3. Select this project and use the following settings:
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`

### GitHub Pages
1. Install the gh-pages package: `npm install gh-pages --save-dev`
2. Add deployment scripts to your `package.json`.
3. Run `npm run deploy`.
